What is the Tech Radar?

Implex1v Tech Radar is a list of technologies, complemented by an assessment result, called ring assignment. I use the four rings with the following semantics:

  • ADOPT — TODO
  • TRIAL — TODO
  • ASSESS — TODO
  • HOLD — TODO

What is the purpose?

The Tech Radar is a tool to visualize my current knowledge status in regard to the four quadrants. This is not meant to be a recommendation, but simply my current state.

BTW, if you would like to create your own Tech Radar — the code is open sourced by Zalando.
